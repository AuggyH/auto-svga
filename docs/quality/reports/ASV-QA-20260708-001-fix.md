## Root Cause

- Preview playback set the canvas intrinsic bitmap size from SVGA metadata, but the visible CSS box still used square sizing rules.
- Wide movie sizes such as `981 x 360` could therefore be displayed inside a square visual box and appear squeezed during playback or after window resizing.
